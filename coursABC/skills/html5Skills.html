Introduction à HTML5

Le cours de HTML5 pour comprendre les bases du langage et les balises essentielles.

Comment structurer son HTML

Il est important de bien structurer son HTML pour le rendre accessible, facile à lire et conforme au SEO.

  • Ajouter votre meta description, meta viewport pour le responsive et votre title
  • Dans le <body> découper votre contenu en 3 parties : <HEADER>, <MAIN>, <FOOTER>
  • Dans le <MAIN>, utiliser des sections pour chaque partie de votre contenu
  • Attention à un bien fermer les balises

Les média en HTML5

Pour les media utilisez les balises <IMG> & <VIDEO>, pour les textes balises <P>, <SPAN> ou <STRONG>, les titres, les balises <H1>,<H2>,<H3>,<H4>

Vidéo

Pour intégrer une vidéo, utilisez la balise <video> avec les attributs controls, autoplay, et muted.

  • Attributs de la balise <video> :

  • controls : Affiche les contrôles de lecture
  • autoplay : Démarre la vidéo automatiquement
  • muted : Coupe le son de la vidéo
  • loop : Rejoue la vidéo en boucle

Note : Assurez-vous que le fichier vidéo est au format MP4 pour une compatibilité maximale avec les navigateurs.

Vous pouvez aussi intégrer des vidéos YouTube en utilisant la balise <iframe>. Allez votre video Youtube et cliquer sur "Partager", puis sur "Intégrer". Copiez le code fourni et collez-le dans votre page.

Image

Pour intégrer une image, utilisez la balise <img> avec l'attribut src pour spécifier le chemin de l'image.

  • Attributs de la balise <img> :

  • src : Chemin de l'image
  • alt : Texte alternatif pour l'image

Note : Si vous faites un site pour la production que ce soit personnel ou professionnel veuillez à faire attention à la licence des vos images

Pour utiliser des images libres de droits vous avez : Pixabay, Unsplash, Pexels

Titres

Utilisez les balises de titre <h1> à <h6> pour structurer votre contenu.

<h1>Titre principal

<h2>Sous-titre

<h3>Titre de section

<h4>Titre de sous-section

<h5>Titre de niveau 5

Texte

Utilisez les balises <p> pour les paragraphes, <span> pour le texte en ligne, et <strong> pour mettre en évidence du texte.

<p>Ceci est un paragraphe de texte.

<span>Ceci est un texte en ligne. <strong>Ceci est du texte en gras.

Les tableaux

Utilisez la balise <table> pour créer des tableaux, avec les balises <thead>, <tbody>, et <tr> pour les lignes.

Balise Description
video Pour intégrer des vidéos
audio Pour intégrer des fichiers audio
img Pour intégrer des images

Liens & ancrages

Utilisez la balise <a> pour créer des liens hypertextes. L'attribut href spécifie l'URL du lien.

Documentation HTML5

Le lien s'ouvre dans un nouvel onglet grâce à l'attribut target="_blank".

Pour créer des liens vers des sections de la même page, utilisez des ancrages avec l'attribut id.

Exemple de lien vers la section Introduction à HTML5.

Pour créer un lien vers une adresse email, utilisez le format mailto: dans l'attribut href.

Exemple : Envoyer un email.

Les formulaires

Utilisez la balise <form> pour créer des formulaires, avec les balises <input>, <label>, et <button>.
